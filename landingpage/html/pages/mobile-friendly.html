Tip 1Installeer de WPtouch plugin

Een mobielvriendelijke website is tegenwoordig erg belangrijk aangezien Google tegenwoordig alleen nog maar kijkt naar je mobiele website. Redenen genoeg dus om ervoor te zorgen dat je mobiele website in orde is. WPtouch is een plug-in voor je WordPress website die je website automatisch responsive (mobiel vriendelijk) maakt.
